## Build
To build the project, just execute the following command line :

```bash
$> build.sh a
```
## Run

To execute the build project, just run it with :

```bash
$> build.sh r
```

or you can execute the command line :

```bash
$> java -jar target/Test001-1.0.0.jar
```
